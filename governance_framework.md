# AI Governance Framework

## Controls
- Human review for high-impact decisions
- Source traceability for generated responses
- Evaluation set for accuracy and hallucination checks
- Bias and fairness review where people decisions are involved
- Data privacy and access control checks
- Monitoring of quality, adoption and exception rates

## Metrics
- Manual effort saved
- Review time reduction
- Accuracy / quality score
- User adoption
- Escalation rate
- Exception rate
- Governance incidents
